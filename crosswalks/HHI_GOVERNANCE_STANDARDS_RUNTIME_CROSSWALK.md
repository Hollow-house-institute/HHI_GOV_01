# HHI Governance Standards Runtime Crosswalk

## Purpose

This document operationalizes interoperability between HHI execution-time governance infrastructure and external governance standards.

The objective is runtime governance alignment across:

- NIST AI RMF
- ISO/IEC 42001
- EU AI Act
- OECD AI Principles
- CAISI
- Human-in-the-Loop oversight
- Continuous Assurance systems

This is not a comparison document.

This is governance interoperability infrastructure.
